# UI/UX and Art Direction

## Experience Goals
- Clear, readable progression decisions in short sessions.
- Strong atmosphere without sacrificing legibility.
- Consistent identity across browser, Steam, and Android.

## UX Direction
- Desktop-first responsive layout.
- Mobile adapts through panel stacking and prioritized action bars.
- Keep mission/combat preparation and inventory operations under 3 taps/clicks from home screen.

## Core Screen Set
1. Home hub (timers, quick claims, activity launch).
2. Character (stats, class, loadout).
3. Inventory grid + stash tabs.
4. Shop (rotating stock, purchase actions).
5. Missions (selection, stamina cost, rewards preview).
6. Combat view (turn order, timer, action selection).
7. Jobs (start/collect timed activities).
8. Events (biweekly objectives and rewards).

## Combat UI Principles
- Prominent turn order lane.
- Server timer always visible with drift-safe updates.
- Pre-select action/target with clear "queued" state.
- If no input, communicate fallback auto-action clearly.

## Art Direction
Theme: semi-dark fantasy with broad appeal. Think - elder scrolls.

Reference tone:
- grounded medieval fantasy
- restrained magic highlights
- weathered materials, steel/leather/stone motifs

Visual style:
- 2D painted panels, icons, and item art.
- Character portraits over full 3D rendering.
- Limited animation overlays for hits, buffs, and status changes.

## Color and Lighting
- Base palette: muted stone, slate, iron, parchment.
- Accent palette: class-coded restrained highlights (crimson, arcane teal, amber).
- Contrast standard: maintain text readability in low-saturation dark panels.

## Accessibility Baseline
- Text and UI contrast meeting WCAG AA where practical.
- Distinct icon + color coding (not color-only signaling).
- Animation reduction option.
- Font scaling support for core HUD and inventory labels.

## Asset Production Constraints
- Target resolution tiers for desktop/mobile to minimize memory spikes.
- Icon set prepared as vector-friendly master with raster exports.
- Reuse frame/panel kits for fast content expansion.
